ELO Rankings

This ranks competitors using the ELO ranking system which ranks competitors based on their placements in competitions. A more detailed explanation can be found here: Explanation.
There are different rankings for different events and an overall ranking which takes the average ELO score for each event.
The raw data containing the ELO points for every competitior in the WCA (as one large CSV file) can be downloaded here: ELO File.

Relay Table

This ranks competitors that have an official single and average in 2x2-7x7 by the sum of their official singles and averages.
The raw data (as CSV files) can be downloaded here: Single, Average.
